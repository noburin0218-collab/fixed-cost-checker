# 収益化ガイド（アフィリエイト・有料PDFの導線設定）

このツールは「無料診断で集客 → 各項目のアフィリエイト＆有料PDFで収益化」する設計です。
リンクは **`config.js` の1ファイルを編集するだけ** で全画面に反映されます。

> ⚠️ アフィリエイトリンクは**あなた自身のASPアカウントで発行される固有URL**です。
> 他人のリンクや汎用URLでは報酬が発生しません。下記ASPに無料登録し、各案件で発行した
> 「広告リンク（テキストリンク）」のURLを `config.js` に貼り付けてください。

---

## 1. まず登録する無料ASP（アフィリエイト・サービス・プロバイダ）

匿名・顔出しなしで始められ、本ツールのテーマ（固定費・家計）と相性が良いもの。
複数登録して、案件ごとに条件の良いASPを選ぶのが定石です。

| ASP | 特徴 | 向いている案件 |
|---|---|---|
| **A8.net** | 国内最大手。サイト審査がゆるく初心者向け。案件数が圧倒的。 | 格安SIM・電力/ガス比較・保険・車 ほぼ全部 |
| **もしもアフィリエイト** | W報酬・かんたんリンク。初心者に親切。 | 通信・各種比較 |
| **バリューコマース** | 大手・優良案件。金融/保険に強い。 | 保険・カード・車 |
| **afb（アフィビー）** | 美容/金融に強く報酬が高め。 | 保険・通信 |
| **アクセストレード** | 通信・金融に強い。 | 格安SIM・電力・カード |

> 登録には「サイト（このGitHub PagesのURL）」が必要です。先に公開してから登録するとスムーズです。
> 報酬の受け取りには本人名義の銀行口座が必要です（運営自体は匿名で可能）。

---

## 2. カテゴリ別：探すべき案件（`config.js` の対応キー）

`config.js` の `affiliates` の各 `href` に、発行したリンクを貼ります。

| config キー | 画面の項目 | ASPで検索するキーワード例 |
|---|---|---|
| `mobile` | スマホ代 | 「格安SIM」「楽天モバイル」「mineo」「IIJmio」「LINEMO」 |
| `electricity` | 電気代 | 「電気 比較」「エネチェンジ」「新電力 切り替え」 |
| `gas` | ガス代 | 「ガス 比較」「プロパンガス 見積もり」「ガス 乗り換え」 |
| `insurance` | 保険料 | 「保険 無料相談」「ほけんの窓口」「保険見直しラボ」「マネードクター」 |
| `car` | 車関連費 | 「自動車保険 一括見積もり」「保険スクエアbang」「インズウェブ」 |

> ※ 案件の有無・報酬額・提携可否はASP／時期により変わります。提携承認後にリンクを発行してください。
> ※ 金融商品（保険）の表現は景表法・薬機法ならぬ各種ガイドラインに注意。「必ず安くなる」等の断定は避ける（本ツールも免責を明記済み）。

## 3. 有料PDF / チェックリスト（自社デジタル商品）

`config.js` の `cta.href` に販売ページURLを設定します。

- **販売プラットフォーム**：note（記事＋有料エリア）、Tips、BOOTH など。匿名販売・低コストで開始可能。
- **商品例**：「固定費見直し完全チェックリスト（PDF）」「我が家の固定費シミュレーションテンプレ（スプレッドシート）」
- **価格帯**：まずは 0〜500円で実績作り → レビュー獲得後に値上げ。

---

## 4. 設定例（`config.js`）

```js
window.SITE_CONFIG = {
  gaMeasurementId: "G-ABCDE12345",
  cta: {
    label: "固定費見直しチェックリスト（PDF）を受け取る",
    href: "https://note.com/yourname/n/xxxxxxxxxxxx",
  },
  affiliates: {
    mobile:      { label: "格安SIMのプランを比較する",     href: "https://px.a8.net/...（発行したリンク）" },
    electricity: { label: "電気・ガス料金を比較する",       href: "https://px.a8.net/..." },
    gas:         { label: "ガス会社を比較・相見積もりする", href: "https://px.a8.net/..." },
    insurance:   { label: "保険の無料相談を予約する",       href: "https://px.a8.net/..." },
    car:         { label: "自動車保険を一括見積もりする",   href: "https://px.a8.net/..." },
  },
};
```

`href` が空欄（`""`）のままの項目は、診断結果にボタンが表示されません（レイアウトは崩れません）。
リンクを取得できたものから順に埋めていけばOKです。

## 5. 効果測定（任意）

`config.js` の `gaMeasurementId` に Google Analytics 4 の測定ID を設定すると、
次のイベントが自動で計測されます。どの導線が効いているかを見ながら改善できます。

- `diagnose`（診断実行／世帯人数・契約形態・削減額つき）
- `share`（X / LINE / コピー）
- `share_image`（結果画像の保存・共有）
- `cta_click`（有料PDFボタンのクリック）
